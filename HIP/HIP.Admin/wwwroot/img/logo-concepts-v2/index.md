# HIP Logo Concepts V2

## 01 — Shield Lockup (Icon + Wordmark)
- **Meaning:** Shield + lock clearly communicates protected identity and secure access.
- **Intended usage:** Primary brand lockup for app header, login screens, sales pages, and docs.

## 02 — Hex Shield Check (Icon + Wordmark)
- **Meaning:** Structured security badge with checkmark signals verified and trusted identity.
- **Intended usage:** Compliance-facing pages, trust messaging, and enterprise collateral.

## 03 — Identity Badge Shield (Icon + Wordmark)
- **Meaning:** ID card inside a shield emphasizes identity-first protection.
- **Intended usage:** Product UI branding where identity workflows are central.

## 04 — Secure Core Shield (Icon-only)
- **Meaning:** Compact shield-lock icon for strong at-a-glance security recognition.
- **Intended usage:** Favicons, app icons, side-nav compact logo, and small-space UI marks.

## 05 — Verified Identity Badge (Icon-only)
- **Meaning:** Person silhouette plus verification check denotes authenticated, trusted users.
- **Intended usage:** Status chips, feature badges, security indicators, and secondary brand marks.
